# Планы по развитию проекта (Roadmap)

В этом файле фиксируются долгосрочные задачи по улучшению инфраструктуры и функционала SEO Pulse.

## 🛠 Инфраструктура и ИИ-инструментарий

### 1. Интерактивный Telegram-бот (PTY Mode)
Сейчас бот работает в режиме YOLO (`-y`), что позволяет выполнять задачи автономно, но не дает контроля над критическими изменениями.
- **Задача**: Переписать `scripts/telegram-bot.js` с использованием библиотеки `node-pty`.
- **Функционал**:
  - Бот должен эмулировать настоящий терминал.
  - При появлении запроса на подтверждение (`Confirm action? [y/n]`), бот должен отправлять в Telegram сообщение с **Inline-кнопками** (✅ Одобрить / ❌ Отклонить).
  - Реализовать потоковую передачу логов (Streaming) вместо ожидания завершения команды.
- **Преимущества**: Безопасность при удаленном управлении и лучший пользовательский опыт.

### 2. Расширенная система метрик
- **Задача**: Добавить визуализацию затрат (токенов и времени) в админ-панель проекта.
- **Функционал**: Графики потребления API Gemini и расчет эквивалентной стоимости работы программиста на основе данных из `agent_tasks`.

## 📈 SEO Функционал
- **Задача**: Реализовать периодический автоматический аудит проектов по расписанию (Cron).
- **Функционал**: Бот должен присылать отчет в Telegram, если показатели SEO (например, скорость загрузки или доступность) ухудшились.
